# Web Scrapper

A Chrome DevTools extension that clones websites, capturing network requests, DOM snapshots, assets, API responses, and storage.

## Features
- Capture Network Requests
- Capture DOM Snapshots
- Capture Assets
- Capture API Responses
- Capture Storage

## Installation
- Open Chrome and navigate to `chrome://extensions/`
- Enable "Developer mode"
- Click "Load unpacked" and select the `voxyz-cloner-extension` directory
